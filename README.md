# raya-plugins

Plugins and integrations for LiveKit and other services.

## Layout

| Path | Purpose |
|------|---------|
| [`python/`](python/) | Python packages (one subdirectory per installable project). |
| [`node/`](node/) | Future npm / TypeScript packages. |

Python packages each have their own `README.md`.

| Package | Description |
|---------|-------------|
| [`livekit-plugins-bakbak`](python/livekit-plugins-bakbak/) | **LiveKit Agents** TTS ([Bakbak / Raya](https://docs.litwizlabs.com/documentation/tts-overview)). `pip install livekit-plugins-bakbak` — see [plugin README](python/livekit-plugins-bakbak/README.md). |
