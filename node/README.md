# Node / TypeScript packages

This folder is reserved for future npm packages (for example LiveKit Agents JS plugins or other service clients).

Suggested layout when you add a package:

```text
node/
  livekit-plugin-bakbak/   # or @your-scope/package-name
    package.json
    src/
```

Keep each package in its own subdirectory with its own `package.json` and lockfile.
